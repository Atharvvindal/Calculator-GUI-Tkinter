# 🧮 Calculator GUI (Tkinter)

A desktop calculator built using Python and Tkinter. The application provides a clean graphical interface for performing common arithmetic operations.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Modulus
- Exponentiation
- Backspace
- Clear display
- Error handling
- Responsive button layout

## Technologies Used

- Python 3
- Tkinter

## How to Run

```bash
python calculator.py
```
